# Flavorly 🥘

**Flavorly** is a vibrant, budget-conscious Android application designed for Kenyan youth and culinary enthusiasts. It provides a curated collection of authentic Kenyan recipes with estimated costs in KES to help users plan affordable and delicious meals.

## 🚀 Features

- **Authentic Kenyan Recipes**: Pre-loaded with favorites like Ugali Sukuma, Chapati Dondo, Smokie Pasua, and more.
- **Budget-Friendly Planning**: Every recipe includes an estimated cost to help you stay within your budget.
- **Horizontal Categories**: Quickly filter recipes by "Quick Meals", "Budget Friendly", "Traditional", and "Special".
- **External Exploration**: Categories link to popular Kenyan food blogs for expanded culinary inspiration.
- **Dynamic UI**: Features a custom "What's Cooking?" animation and a modern, blended aesthetic.
- **Real-time Backend**: Powered by **Supabase** for seamless data management.

## 🛠️ Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose (Material 3)
- **Backend**: Supabase (Postgrest & Auth)
- **Networking**: Ktor
- **Architecture**: MVVM (Model-View-ViewModel)
